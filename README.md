## Tech stack
- Solidity v0.6.12
- Truffle v5.1.52
- web3.js v1.2.1
- Node.js v11.15.0

&nbsp;

***

## Setup  
### ① Install modules  
```
$ npm install
```

<br>

### ② Run ganache-cli  
（Please make sure whether port number is `8545` or not）  
```
$ ganache-cli
```

<br>

### ③ Compile & migrate contracts  
```
$ npm run migrate:local
```

<br>

### ④ Test contracts  
```
$ npm run test:local
```
